# Vertigo Automation suite:

Vertigo A.S is a high-speed, offline-first management suite built to guarantee 100% operational uptime for retail and local businesses, even during severe network outages.

This repository contains the interactive architecture presentation and UX prototype built using Streamlit.

## 🚀 Live Demo
👉 https://vertigosoftware-demo-q9pah3xyuyvkqkvbv8wact.streamlit.app/

## 🛠️ Core Architectural Pillars
* **Zero-Latency Offline Writes:** Designed to bypass flaky internet infrastructure by routing database transactions locally via SQLite before syncing upstream.
* **Smart Budget Automation:** Dynamic data rendering engines built to track cash flows and monitor category allocations instantly.
* **Simulated Resiliency Engine:** An interactive simulation module demonstrating automatic cloud background-sync loops once connectivity is restored.

## 💻 Tech Stack
* **Language:** Python
* **Interface Engine:** Streamlit
* **Database Management:** SQLite3 (Standard Library Engine)
